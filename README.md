Objective :-

To learn how Git works with branches and how merge conflicts are resolved.

What I Did?
* Created a Git repository using git init
* Added a file (file.txt) and made the first commit
* Created a new branch called feature
* Made changes to the file in the feature branch and committed
* Switched back to the main branch and made different changes
* Tried to merge both branches.

Merge Conflict :-
A conflict occurred because the same file was changed differently in both branches, and Git could not decide which version to keep.

Resolution :-
* Opened the file and checked both versions
* Removed the conflict markers
* Wrote a final combined version
* Saved and committed the resolved file

Conclusion :-
This task helped me understand how Git handles conflicts and how to resolve them manually.
